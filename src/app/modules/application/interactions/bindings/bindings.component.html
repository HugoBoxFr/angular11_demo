Bindings

In Angular, a template is a chunk of HTML. Within a template, you can use special syntax to leverage many of Angular's features."

Angular.io
Property binding

Uses [ ] on html properties or directives to set values.

app.component.html

<button [disabled]="buttonDisabled" class="btn btn-success">Example</button>

app.component.ts

buttonDisabled = true;

Event binding

Uses ( ), allows to listen to user actions like click, mousenter...

app.component.html

<button (click)="onSwich()" class="btn btn-warning">Switch</button>

app.component.ts

light = false;

onSwitch() {
this.light = !this.light;
}

Two-way binding

Uses [( )] (banana in a box). Binds template and code.

import { FormsModule } from '@angular/forms';

app.component.html

<input class="form-control" [(ngModel)]="text">
