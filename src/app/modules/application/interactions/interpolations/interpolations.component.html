Interpolations

"Dynamically change what appears in an application view, such as displaying a custom greeting that includes the user's name."

- Angular.io -

Hello Stranger... What's your name ?
Great to see you again {{ name }} !

Angular interpolation is used to display a component property in the respective view template with double curly braces syntax. You can display all kind of properties data into view e.g. string, number, date, arrays, list or map.

Example: { { property } }
